 # Support

If you need help using OfferZone or want to discuss usage, please use one of the following channels:

- **Issues:** Use GitHub Issues for bug reports and feature requests (non-sensitive).
- **Discussions:** (If enabled) use GitHub Discussions for broader questions and community conversation.
- **Email / Commercial Support:** For private questions or commercial support, contact the maintainers listed in `MAINTAINERS.md` or a support address if provided.
